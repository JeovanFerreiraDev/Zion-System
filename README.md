# 🏛️ Zion System

O **Zion** é uma fortaleza digital para a gestão de dados escolares. Desenvolvido para operar exclusivamente em ambiente de rede local (Intranet), o sistema garante que informações sensíveis de alunos, livros e mensalidades permaneçam protegidas e acessíveis apenas a usuários autorizados.

Este projeto é um pilar fundamental nos meus estudos de **Engenharia de Software na Anhanguera**.

---

## 🛡️ Arquitetura e Segurança

* **Núcleo:** Python com o framework Flask.
* **Armazenamento:** SQLite, garantindo uma base de dados leve e de rápida resposta local.
* **Protocolo de Acesso:** Autenticação restrita por senha e filtragem de conexão por rede interna, assegurando a integridade dos registros.

---

## ✨ Módulos do Sistema

* 📑 **Registro de Alunos:** Gestão centralizada de inscrições e perfis acadêmicos.
* 💰 **Gestão Financeira:** Acompanhamento de mensalidades e status de pagamentos de forma segura.
* 📖 **Acervo Escolar:** Controle de livros e materiais didáticos vinculados aos estudantes.
